EXPLICAÇÃO SOBRE O TRABALHO

Este projeto visou a criação de um site utilizando HTML, CSS e JavaScript, com um foco especial na aplicação e avaliação das funcionalidades JavaScript. A estrutura do site foi organizada de forma clara e funcional, seguindo a arquitetura especificada

Cada página possui um menu customizado e original, garantindo uma navegação intuitiva entre as diferentes funcionalidades:

Página Lista:

Apresenta uma tabela com quatro colunas: Nome, Email, Data de Nascimento e Botão Deletar. O JavaScript foi utilizado para adicionar a funcionalidade do botão deletar, que ao ser clicado, esconde a linha correspondente da tabela. Esta funcionalidade demonstra a manipulação do DOM (Document Object Model) e eventos em JavaScript.

Página API:

Realiza uma chamada à API “https://jsonplaceholder.typicode.com/posts/1” e exibe o resultado na tela. Aqui, o JavaScript mostra sua capacidade de interagir com APIs externas através do método fetch, manipulando e apresentando dados dinâmicos na interface do usuário.

Página Botão:

Contém três botões ("Botão 1", "Botão 2", "Botão 3") e três divs de texto ("Div 1", "Div 2", "Div 3") que começam escondidas. A lógica em JavaScript faz com que ao clicar em um botão, a div correspondente seja exibida e as outras sejam escondidas. Isso evidencia o uso de eventos e manipulação de estilos para controlar a visibilidade de elementos no DOM.

Considerações Finais

A implementação deste projeto demonstra uma compreensão sólida de conceitos essenciais de desenvolvimento web, incluindo: Estruturação de arquivos e organização do projeto. Criação de interfaces responsivas e interativas utilizando HTML e CSS. Aplicação de JavaScript para manipulação do DOM, eventos, e interação com APIs externas. O trabalho não só atende aos requisitos especificados, mas também proporciona uma base sólida para futuras expansões e melhorias. A abordagem modular facilita a manutenção e escalabilidade do projeto. Em suma, o projeto ilustra com sucesso o uso integrado de HTML, CSS e JavaScript para criar uma aplicação web funcional e interativa.
